Swara | Software Developer Portfolio

A modern, interactive personal portfolio built using React, TypeScript, and Three.js.

This project combines clean UI with immersive 3D visuals to showcase my skills, projects, and experience as a Full Stack Developer.

🎯 About the Project

This portfolio is designed to stand out with interactive 3D elements powered by Three.js while maintaining performance and responsiveness.

It highlights:

My development skills (Frontend + Backend)

Real-world projects

Experience and achievements

Contact and professional links


The goal: not just a portfolio, but an interactive experience.

🛠️ Tech Stack

Frontend: React.js

Language: TypeScript

3D Graphics: Three.js

React 3D Renderer: React Three Fiber 

Styling: CSS3 / Tailwind CSS / SCSS

Build Tool: Vite

Deployment: Vercel 


✨ Features

⚡ Responsive modern UI (mobile + desktop)

🎮 Interactive 3D scene using Three.js

🧠 Type-safe development with TypeScript

🧩 Modular React component architecture

🎨 Smooth animations and transitions

📂 Projects showcase with visual appeal

👤 About, Skills, Experience sections

📬 Contact section with social links

🚀 Optimized performance for 3D rendering

⚙️ Getting Started (Run Locally)

Clone the repository:

git clone https://github.com/swara03965/Portfolio_Swara.git

Navigate into the project:

cd Portfolio_Swara

Install dependencies:

npm install

Start development server:

npm run dev
